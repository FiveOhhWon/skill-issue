Architecture

Three Surfaces, One Skill Library

Every skill works identically whether invoked from Claude Code, Slack, or the CLI. One shared library, three access patterns.

Claude Code

Power users invoke skills as plugin commands. Full pipeline orchestration with typed outputs, agents, and subagents.

Slack Bot

Non-technical users compose skills through natural language. Streaming responses, progress indicators, and threaded conversations.

CLI Skillkit

Generate new agentskills.io-compliant skills from natural language descriptions. Validate, test, and deploy in one command.

Capabilities

Built for Real Operations

Every feature designed for production newsletter workflows, not toy demos.

agentskills.io Standard

Open, cross-platform skill format. SKILL.md with YAML frontmatter. Works with any AI agent runtime that supports the standard.

Composable Pipelines

Skills chain via typed contracts. Outputs from one skill flow automatically as inputs to the next through prompt-driven orchestration.

Slack Integration

Non-technical users access the full skill library via natural language. Streaming responses, progress indicators, and session continuity.

HubSpot CRM

End-to-end business operations. Sponsor proposals automatically create deals, log notes, and update contacts in HubSpot via MCP.

Skillkit

Generate new skills from plain English. The multi-agent CLI plans, writes, and validates agentskills.io-compliant skills automatically.

TypeScript Throughout

Full type safety, zero Python. Zod schemas validate at application boundaries. TypeScript handles analytics, orchestration, and all compute.

Quickstart

Up and Running in Minutes

Clone, install, and start composing skills in three steps.

1

Clone & Install

Clone the monorepo and install all workspace dependencies with a single command. TypeScript builds automatically.

2

Load the Plugin

Open Claude Code with the plugin flag. All six skills and three pipeline commands are available immediately.

3

Run a Pipeline

Invoke a command to run a full pipeline. Skills compose automatically, each feeding typed output to the next.

terminal bash
# Step 1: Clone & install
$ git clone https://github.com/levi/skill-issue.git
$ cd skill-issue && npm install
 
# Step 2: Load the plugin in Claude Code
$ claude --plugin ./plugin
 
# Step 3: Run a pipeline
> /daily-brief --topic "AI agents"
Researching... Analyzing... Generating...
Brief complete (5 stories, 12 sources)
 
# Or generate a new skill
$ npx skillkit --describe "Track podcast mentions" --name podcast-tracker
Planning... Writing... Validating...
Skill created at skills/podcast-tracker/SKILL.md
