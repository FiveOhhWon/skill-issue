## Direction
- Retro desktop OS shell with tactile window chrome and icon rails.
- Warm paper palette, ink outlines, orange/teal accents.
- Two-window composition:
  - Main: `Newsletter Command Deck` dashboard with KPI blocks, timeline, trend chart, queue.
  - Side: `ops-chat.ai` panel with visible streaming state and markdown-like list output.

## Sensible Defaults Used
- Desktop-first layout, responsive stack below 1280px.
- Typography: `Chivo` (UI) + `Space Mono` (status/technical bits).
- Animation: icon hover float, chat stream typing cursor.
- No production app components modified yet; this is a standalone concept artifact.
